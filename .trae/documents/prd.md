## 1. 产品概述

配色渐变与 Shadow 设计令牌工具——面向 UI 设计师与前端工程师的可视化设计令牌生成平台。输入品牌主色，一键生成完整色板阶梯、配色方案、渐变、阴影与玻璃态参数，实时预览组件效果并导出多格式代码。

- 解决设计与开发之间色值传递的断层问题，将配色决策从"凭感觉"变为"系统化"
- 目标用户：独立开发者、小型团队、设计系统维护者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|---------|---------|
| 设计师/开发者 | 无需注册，直接使用 | 全部功能 |

### 2.2 功能模块

1. **首页（/）**：色板阶梯 + 预览面板分屏，主色输入、阶梯生成、配色方案、色盲模拟
2. **渐变页（/gradient）**：渐变编辑器，多色 stop 拖拽、角度调节、线性/径向切换、预设
3. **阴影页（/shadow）**：阴影编辑器，层叠阴影参数、预设、玻璃态参数
4. **导出页（/export）**：汇总导出，CSS 变量 / Tailwind 配置 / tokens.json / 剪贴板

### 2.3 页面详情

| 页面 | 模块 | 功能描述 |
|------|------|---------|
| 首页 | 主色输入 | HEX 输入框、颜色选择器、图片取色（canvas quantize） |
| 首页 | 色板阶梯 | 输入 brand HEX → 生成 50–950 共 11 级 palette（OKLCH 算法）；每级可复制色值；白/黑文字对比度 pass/fail 标注 |
| 首页 | 配色方案 | 类似、互补、三角、四角方案；色轮交互选主色；色盲模拟（protan/deutan）预览 |
| 首页 | 预览面板 | 亮/暗背景切换；示例 UI：主按钮、次要按钮、输入框、卡片、Alert |
| 首页 | 项目管理 | localStorage 保存多套 Palette Project；命名切换 |
| 渐变页 | 渐变编辑器 | 多色 stop 拖拽条；角度滑块；linear / radial 切换；CSS 代码实时输出；预设 aurora/sunset 等 6 套 |
| 阴影页 | 阴影编辑器 | 层叠阴影列表：x/y/blur/spread/color/inset；预设 sm/md/lg/elevated；浮动卡片预览 |
| 阴影页 | 玻璃态编辑器 | backdrop-blur 滑块；半透明背景色；边框高光；导出 utility class 片段 |
| 导出页 | 导出汇总 | CSS :root 变量、tailwind.config theme.extend.colors、tokens.json；剪贴板一键复制 |
| 导出页 | 分享链接 | LZ-string 编码 project 数据至 URL，可分享还原 |
| 导出页 | Typography 占位 | Typography scale 占位区域（预留） |

## 3. 核心流程

用户输入品牌主色 → 系统自动生成 11 级色板阶梯 → 用户浏览配色方案并选择 → 在渐变/阴影编辑器中微调 → 预览面板实时查看组件效果 → 导出多格式设计令牌代码。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色工具风（深灰/近黑底色），accent 使用用户自定义的品牌色高亮
- 次要色：中性灰阶，配合亮度层次
- 按钮风格：圆角微凸起，hover 时品牌色发光
- 字体：JetBrains Mono（代码/数值）+ Sora（UI 标题/正文），几何感与科技感
- 布局：左右分屏（编辑区 | 预览区），顶部导航切换页面
- 图标：Lucide 线性图标

### 4.2 页面设计概览

| 页面 | 模块 | UI 元素 |
|------|------|---------|
| 首页 | 主色输入区 | HEX 输入框 + 原生颜色选择器 + 拖拽上传图片区，品牌色大色块展示 |
| 首页 | 色板阶梯 | 11 个水平色块条，每块显示色值 + 对比度标记 + 复制按钮 |
| 首页 | 配色方案 | 色轮 SVG + 方案色块行 + 色盲模拟切换标签 |
| 首页 | 预览面板 | 右侧固定面板，包含按钮组/输入框/卡片/Alert 组件示例 |
| 首页 | 项目管理 | 顶部下拉选择器 + 新建/删除/重命名 |
| 渐变页 | 渐变编辑器 | 大尺寸渐变预览区 + stop 拖拽条 + 角度旋钮 + 类型切换 + 预设卡片网格 + CSS 代码块 |
| 阴影页 | 阴影编辑器 | 浮动卡片预览 + 阴影层列表（可增删排序）+ 参数滑块组 + 预设按钮组 |
| 阴影页 | 玻璃态编辑器 | 磨砂玻璃卡片预览 + blur/透明度/边框滑块 + CSS 代码块 |
| 导出页 | 导出汇总 | 三栏 Tab（CSS/Tailwind/JSON）+ 代码高亮块 + 一键复制按钮 + 分享链接输入框 |

### 4.3 响应式

- 桌面优先：左右分屏布局（编辑区 60% | 预览区 40%）
- 平板：上下堆叠布局
- 移动端：全宽编辑区，预览区折叠为底部抽屉
